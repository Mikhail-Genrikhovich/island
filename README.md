# 🏝️ Island Simulation
**Симуляция экосистемы острова**

Многопоточная симуляция жизни острова с животными и растениями:  
движение · питание · размножение · рост растений · смерть · статистика

## 1. Структура проекта

### 1.1 model — модельные классы

- **Animal** — абстрактный базовый класс всех животных
- **Predator** — абстрактный класс хищников
- **Herbivore** — абстрактный класс травоядных
- Конкретные животные:  
  Wolf · Boar · Rabbit · Mouse · …
- **Plant** — класс растения
- **Island** — остров (двумерный массив `Location[][]`)
- **Location** — клетка острова  
  └─ содержит: список животных + одно растение

### 1.2 service — логика симуляции

- **InitialPopulationService** — случайное начальное заселение острова
- **MoveService** — движение животных
- **EatService** — процесс питания (хищники + травоядные)
- **ReproduceService** — размножение животных
- **ResetReproduceService** — сброс флага `hasReproducedThisTurn`
- **DeathService** — удаление погибших животных
- **PlantGrowService** — рост растений на клетках
- **StatisticsService** — сбор статистики и проверка условий завершения

### 1.3 simulation — ядро симуляции

- **SimulationEngine**  
  Многопоточный движок на базе `ScheduledExecutorService` + `ExecutorService`
